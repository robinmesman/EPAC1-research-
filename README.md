This repository contains all protein structures generated by RoseTTAFoldDiffusion 3 (RFD3) under the different design conditions used throughout this project. It also includes all structures generated with Boltz-2, together with their corresponding predicted binding scores for both cAMP- and cGMP-bound states.
In addition, the repository provides both the original reference scripts and the finalized, modified versions of the code that were used to generate the structural predictions and analyses. The scripts used to process, analyze, and visualize the predicted outputs are also included.
